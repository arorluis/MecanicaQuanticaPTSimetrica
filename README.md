# MecanicaQuanticaPTSimetrica

Notebooks de Mathematica per trobar les energies dels hamiltonians de la família epsilon.

El notebook `grafica_total_pas_variable.nb` genera la Figura 10 fent ús de la parametrització general.

El notebook `paratabla.nb` genera la Taula 1.

El notebook `grafica_parts_re_im.nb` genera la Figura 9 integrant en un interval real.

També s'hi inclouen les versions en `.txt` que contenen únicament el codi font, sense eixida gràfica ni outputs de Mathematica.
